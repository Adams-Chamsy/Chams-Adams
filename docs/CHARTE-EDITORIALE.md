# Charte Éditoriale — Chams Adams

> Guide de référence pour tout contenu texte du site.

---

## 🎯 Philosophie

Le kaftan Chams Adams n'est pas un vêtement. C'est un **héritage porté**, une **œuvre d'art mobile**, un **langage de grâce**.

Chaque mot écrit sur le site doit porter cette conviction.

---

## ✍️ Voix de la marque

**Ce que nous sommes** :
- Noble sans être hautain
- Poétique sans être obscur
- Enraciné sans être passéiste
- Moderne sans renier l'héritage

**Ce que nous ne sommes pas** :
- Exotique
- Folklorique
- Commercial ou agressif
- Descriptif et plat

---

## 📝 Règles d'écriture

### Longueur
- Titres : 2 à 6 mots maximum
- Sous-titres : une phrase poétique
- Descriptions produits : 3 à 5 phrases, jamais plus
- Manifeste : respire, aère, laisse l'espace parler

### Ton
- Préférer le **présent intemporel** au passé composé
- Utiliser la **3e personne** pour la marque, la **2e personne** rare mais puissante pour le client
- Phrases courtes et longues alternées, comme un rythme musical

### Vocabulaire privilégié
| Oui | Non |
|-----|-----|
| Majesté | Swag |
| Héritage | Tradition |
| Grâce | Style |
| Noblesse | Prestige |
| Lumière | Brillance |
| Transmission | Tendance |
| Présence | Look |
| Œuvre | Produit |
| Pièce | Article |
| Maison | Marque |

### Mots à bannir
- Ethnique
- Exotique
- Tribal
- Authentique (trop cliché)
- Trendy / Branché
- Unique (sauf si vraiment unique)
- Swag / Vibe
- Emojis dans le contenu éditorial

## 🌍 Références culturelles

Les occasions à nommer avec justesse :
- **Mariage** (cérémonie, alliance)
- **Tabaski** (Aïd el-Kébir)
- **Magal** (Touba, Sénégal)
- **Maouloud** (naissance du Prophète)
- **Korité** (Aïd el-Fitr)
- **Baptême**
- **Tamkharit** (Achoura)

Toujours les écrire avec majuscule, comme des noms propres sacrés.
